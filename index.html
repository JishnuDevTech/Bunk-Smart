About Bunk Smart

Bunk Smart is a comprehensive attendance management system designed to help students track their attendance, build discipline, and achieve academic goals through intelligent insights and gamified challenges.

Security

Your privacy and data security are our top priorities. We use Firebase Authentication for secure login with Google and GitHub options. All attendance data is stored securely in Firestore with encryption at rest and in transit. No sensitive information is shared without consent, and we comply with best practices for web security including HTTPS and input validation.

Process

The webpage works seamlessly: Users sign up or log in via the landing page. Once authenticated, they access the dashboard to mark attendance on an interactive calendar, view analytics in the insights section, and participate in challenges. Data is synced in real-time using Firebase, with local storage for offline resilience. Exports are available in CSV/PDF for records.
